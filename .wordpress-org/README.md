# WordPress.org Assets

This directory contains assets for the WordPress.org plugin repository.

## Files

Place the following files in this directory:

- `banner-772x250.png` - Plugin banner (high resolution)
- `banner-1544x500.png` - Plugin banner (retina)
- `icon-128x128.png` - Plugin icon (small)
- `icon-256x256.png` - Plugin icon (large)
- `screenshot-1.png` - First screenshot
- `screenshot-2.png` - Second screenshot
- etc.

These files will be automatically deployed to the WordPress.org SVN `assets` directory when the deploy workflow runs.

## Documentation

See: https://developer.wordpress.org/plugins/wordpress-org/plugin-assets/
